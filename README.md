# WEB_exam_2026

## Экзаменационное задание по дисциплине "Основы веб-технологий"

Сайт языковой школы, в которой студенты могут изучать иностранные языки.

### Структура проекта

- `index.html` - главная страница сайта
- `account.html` - личный кабинет пользователя
- `css/style.css` - основные стили сайта
- `js/main.js` - основной JavaScript для главной страницы
- `js/api.js` - модуль для работы с API
- `js/account.js` - JavaScript для личного кабинета
- `js/map.js` - модуль для работы с интерактивной картой

### Функционал

#### Главная страница
- Шапка сайта с навигацией
- Область уведомлений
- Блок описания языковой школы
- Преимущества изучения иностранного языка (Bootstrap Grid)
- Форма поиска курсов
- Предлагаемые курсы с пагинацией
- Список репетиторов
- Интерактивная карта учебных ресурсов (Яндекс.Карты)

#### Личный кабинет
- Создание новой заявки на курс
- Просмотр списка заявок
- Редактирование заявок
- Удаление заявок

### API

Проект использует REST API для работы с данными:
- `/api/courses` - получение списка курсов
- `/api/course/{id}` - получение информации о курсе
- `/api/tutors` - получение списка репетиторов
- `/api/tutors/{id}` - получение информации о репетиторе
- `/api/orders` - работа с заявками (GET, POST)
- `/api/orders/{id}` - работа с конкретной заявкой (GET, PUT, DELETE)

### Технологии

- HTML5
- CSS3
- JavaScript (ES6+)
- Bootstrap 5.3.0
- Яндекс.Карты API

### Установка и запуск

#### Локальная разработка (с прокси-сервером для обхода CORS)

1. Убедитесь, что у вас установлен Node.js
2. Установите зависимости (если необходимо): `npm install`
3. Запустите прокси-сервер: `npm start` или `node proxy-server.js`
4. Откройте в браузере: `http://localhost:3001/index.html`

Прокси-сервер автоматически перенаправляет все запросы к `/api/*` на реальный API сервер и добавляет необходимые CORS заголовки.

#### Прямое использование (на хостинге)

1. Загрузите файлы на хостинг
2. Откройте `index.html` в браузере
3. API запросы будут выполняться напрямую к `http://exam-api-courses.std-900.ist.mospolytech.ru/api`

### Решение проблемы CORS

При локальной разработке браузер блокирует запросы к API из-за политики CORS. Для решения этой проблемы:

1. **Используйте прокси-сервер** (рекомендуется):
   - Запустите `node proxy-server.js`
   - Откройте `http://localhost:3001/index.html`

2. **Используйте расширение браузера** (только для разработки):
   - Установите расширение "CORS Unblock" или "Allow CORS"
   - Включите его при разработке

3. **На хостинге**: Проблема CORS не возникает, если сайт и API находятся на одном домене или сервер API настроен правильно.
